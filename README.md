# YOLOv8-ECP-Pedestrian-Detection

A multi-model pedestrian detection benchmark and interactive viewer built on the [EuroCity Persons (ECP)](https://eurocity-dataset.tudelft.nl/) dataset.

Fine-tunes YOLOv8s on 238K+ pedestrian instances across 31 European cities and benchmarks 9 models from the YOLO and MobileNet families side-by-side.

---

## Features

- **Interactive viewer** — browse ECP images, run inference, navigate with keyboard shortcuts
- **9 models** — YOLOv8 (n/s/m/l), RT-DETR-l, SSDLite320-MV3, FasterRCNN-MV3 (×2), YOLOv8s fine-tuned
- **Live stats panel** — FPS, inference ms, detection count, device
- **Benchmark tab** — run all models on a val subset, sort results, export CSV
- **VS Code dark theme** — PySide6 UI with consistent styling
- **Background model loading** — UI never freezes during model load

## Setup (Windows · Python 3.11)

**1. Install PyTorch** (do this first — match your CUDA version):

```bat
:: CUDA 12.1
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

:: CUDA 11.8
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

:: CPU only
pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
```

**2. Run setup:**

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Or just double-click `setup.bat`.

## Running the Viewer

```bat
.venv\Scripts\activate
python viewer\app.py
```

Or double-click `run.bat`.

**Keyboard shortcuts:**

| Key | Action |
|---|---|
| `Space` | Run inference |
| `←` / `→` | Previous / Next image |
| `Ctrl+L` | Load selected model |

## Fine-tuning on ECP

**Step 1 — Convert annotations:**

```bat
python training\convert_ecp_to_yolo.py --ecp_root C:\ECP --out_root dataset
```

**Step 2 — Train:**

```bat
python training\train.py --epochs 50 --batch 16 --device 0
```

Best weights saved to `runs/train/yolov8s-ecp/weights/best.pt`.
Load them in the viewer via the "YOLOv8s · ECP fine-tuned" dropdown option.

**Common flags:**

```bat
python training\train.py --batch 8     # reduce if GPU runs out of memory
python training\train.py --resume      # resume from last checkpoint
python training\train.py --model yolov8m.pt  # use a larger base model
```

## CLI Benchmark

```bat
python benchmark\run_benchmark.py --val_root C:\ECP\ECP\val\img --n 200
```

Prints a sorted table and saves a CSV to `benchmark/results/`.
